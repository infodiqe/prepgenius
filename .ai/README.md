# .ai — Machine context for AI coding agents

**Canonical agent rules live at the repository root** as `CLAUDE.md` (Claude Code) and
`AGENTS.md` (Codex / Cursor / OpenCode), because those tools auto-discover rules at root.
Edit those files — do not fork them here.

This folder holds supporting context:
- `rules/development_principles.md` — the shared principles the rules enforce.
- `context/` — project overview, coding standards, and glossary for grounding agents.
- `prompts/` — reusable task prompts for common work.
